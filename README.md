# Content Listener Agent

Claude Code agent that transcribes, summarizes, and extracts actionables from YouTube videos, Instagram reels, and Spotify podcasts. Uses yt-dlp and Whisper for transcription, stores results in SQLite, and leverages Claude Code in-session for free summarization and actionable extraction.

## Usage

This is a Claude Code agent. Add the `.md` file to your `.claude/agents/` directory to use it.
